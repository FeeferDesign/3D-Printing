# Summary

This knob is to make it easier to spin the extruder to load your filament if you are using a manual loading process with a Bowden Setup. I built this for my SeeMeCNC Rostock Max v3 but have many other applications for it.

I used a 10-32 1 1/2 Inch Machine Screw, 2 x 10-32 nuts, and 2 x #8 Washers to install the knob.

Added Files for M5 Hardware.
